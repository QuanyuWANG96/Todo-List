# Todo-List

This todo list will show your tasks with priority. A lower integer indicates a higher priority.(e.g., an item of Pririty 2 is higher priority than an item of priority 5). 

### lINK to vedio demo for this todo list
[link on youtube demo](https://youtu.be/R3Pyhosl9us)


### Manual for Todo List
* You can add your items with the "+" icon and reclick will hide the input box. 
* When the mouse is hovering to the left of this item, it will show a "delete" icon. If you click it, this task will be delete. 
* If you complete this task, you can click it and it will be shown as completed and reclick will make it active again.
